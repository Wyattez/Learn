

## სისტემის განახლება

```bash
apt update
```

ეს ბრძანება აახლებს პაკეტების სიას და ამოწმებს, ხელმისაწვდომია თუ არა ახალი ვერსიები.

---

## MySQL Server-ის ინსტალაცია

```bash
apt install mysql-server -y
```

ეს ბრძანება აყენებს MySQL Server-ს ავტომატური დადასტურებით (`-y`).

---

## MySQL-ში შესვლა Root მომხმარებლით

```bash
mysql -u root
```

ეს ბრძანება ხსნის MySQL-ის კონსოლს `root` მომხმარებლით.

---

## არსებული მონაცემთა ბაზების ნახვა

```sql
SHOW DATABASES;
```

აჩვენებს სერვერზე არსებულ ყველა მონაცემთა ბაზას.

---

## კონკრეტული მონაცემთა ბაზის არჩევა

```sql
USE database_name;
```

ხსნის და ააქტიურებს მითითებულ მონაცემთა ბაზას.

მაგალითი:

```sql
USE school_db;
```

---

## ცხრილების (Tables) ნახვა

```sql
SHOW TABLES;
```

აჩვენებს არჩეულ მონაცემთა ბაზაში არსებულ ყველა ცხრილს.

---

## მუშაობის მაგალითი

```bash
apt update
apt install mysql-server -y
mysql -u root
```

```sql
SHOW DATABASES;

USE school_db;

SHOW TABLES;
```

---

## მოკლე შეჯამება

|ბრძანება|დანიშნულება|
|---|---|
|`apt update`|სისტემის პაკეტების სიის განახლება|
|`apt install mysql-server -y`|MySQL Server-ის ინსტალაცია|
|`mysql -u root`|MySQL-ში შესვლა Root მომხმარებლით|
|`SHOW DATABASES;`|მონაცემთა ბაზების ნახვა|
|`USE database_name;`|კონკრეტული მონაცემთა ბაზის არჩევა|
|`SHOW TABLES;`|ცხრილების (Tables) ნახვა|

```

MySQL-ის ეს ბრძანებები გამოიყენება მონაცემთა ბაზების მართვისთვის, მათ შორის მონაცემთა ბაზების დათვალიერების, არჩევისა და მათში არსებული ცხრილების სანახავად.
```
